arithmetic expression simulator for single digit natural numbers

Documentation

Used Frameworks:

None - The application is writen in plain JS, CSS and HTML.

Implemantation

The was implemanted with JS.

Datastructure

The grammar is internally represented in an array. Which holds all valid states. The generated arithmatic expression is stored in a string.

Algorithm

Until a terminal state in the arithmatic expression string is detected, the application replaces occurences of non terminal symbols with a randomly choosen symbol from the grammar. In a final step all symbols that would lead to a number are replaced with a random number.

Technical particularities:

Due to the fact that the genarator can genarate arbitrarily long arithmatic expressions, I skewed the distribution of possible replacements towards a higher occurence of final states to avoid unnecessarily long terms.
